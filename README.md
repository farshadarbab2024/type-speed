# How to run and test it locally? 
1- clone this repository by running this command: git clone https://github.com/farshadarbab2024/type-speed
2- cd to the project folder by this command: cd type-speed
3- install dependencies by running this command: npm install 
4- Run the project by this command: npm run dev
